HTML entities are special characters that are used to represent characters that have a special meaning in HTML, such as <,>, and &. They are used to display these characters correctly in the browser.

For example, the less than symbol (<) is represented as < in HTML, and the greater than symbol (>) is represented as >. Similarly, the ampersand (&) is represented as &.

Here are some common HTML entities:

  • < - Less than
  • > - Greater than
  • & - Ampersand
  • " - Double quote
  • ' - Single quote
  •   - Non-breaking space
  • © - Copyright symbol
  • ® - Registered trademark symbol
  • € - Euro symbol
  • £ - British pound symbol
  • ¥ - Yen symbol
  • ¢ - Cent symbol
  • § - Section symbol
  • ° - Degree symbol
  • × - Multiplication sign
  • ÷ - Division sign
  • … - Horizontal ellipsis
  • « - Left-pointing double angle quotation mark
  • » - Right-pointing double angle quotation mark
  • • - Bullet point
  • ′ - Prime symbol (minutes or feet)
  • ″ - Double prime symbol (seconds or inches)
  • – - En dash
  • — - Em dash
  • ‘ - Left single quotation mark
  • ’ - Right single quotation mark
  • “ - Left double quotation mark
  • ” - Right double quotation mark
  • · - Middle dot
  • ¹ - Superscript one
  • ² - Superscript two
  • ³ - Superscript three
  • ¼ - Fraction one quarter
  • ½ - Fraction one half

These entities are useful when you want to display these characters in your HTML content without them being interpreted as HTML code.

For example, if you want to display the text "5 < 10" in HTML, you would write it as "5 < 10".

Using HTML entities helps ensure that your content is displayed correctly across different browsers and devices, and it prevents any potential issues with special characters being misinterpreted by the browser.

Here is an example of how to use HTML entities in a paragraph:

<p>This is an example of using HTML entities: < & > " &#39;</p>

When rendered in a browser, it will display as:

This is an example of using HTML entities: < & > " '

Preformatted Text

The <pre> tag is used to display preformatted text in HTML. It preserves the whitespace and line breaks in the text, making it useful for displaying code snippets or other text that requires specific formatting.

<pre>
<h1>This is a heading</h1>
<p>This is a paragraph with preformatted text.</p>
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
