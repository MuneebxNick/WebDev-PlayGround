CSS Media Queries

CSS Media Queries are used to apply styles based on the device characteristics, such as screen size, resolution, and orientation.

Basic Syntax

@media media-type and (condition) {
    /* CSS rules here */
}

Examples

Example 1: Responsive Design

@media screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

Example 2: Print Styles

@media print {
    body {
        font-size: 12pt;
    }
}

Common Media Features

  • width: Width of the viewport.
  • height: Height of the viewport.
  • orientation: Landscape or portrait mode.
  • resolution: Device pixel ratio.

Media Types

  • all: Suitable for all devices.
  • print: For printed documents.
  • screen: For computer screens, tablets, smartphones, etc.
  • speech: For speech synthesizers.

Combining Media Queries

@media screen and (min-width: 600px) and (max-width: 1200px) {
    body {
        background-color: lightgreen;
    }
}
