1. CSS Position Property

Definition: The position property defines how an element is positioned in the document. Common values:

  • static - Default, no special positioning.
  • relative - Positioned relative to itself.
  • absolute - Positioned relative to nearest positioned ancestor.
  • fixed - Stays fixed relative to the viewport.
  • sticky - Toggles between relative and fixed based on scroll.
