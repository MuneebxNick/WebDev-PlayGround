1. CSS Position

The position property determines how an element is placed in the document.
Common values: static, relative, absolute, fixed, sticky.

Relative Box (Parent)
Absolute Box (Child)
<div class="relative-box"> Relative Box (Parent) <div class="absolute-box">Absolute Box</div> </div> .relative-box { position: relative; } .absolute-box { position: absolute; top: 10px; right: 10px; }

2. CSS Z-Index

z-index controls the stack order of elements. Higher values are displayed on top of lower ones.

Box 1
Box 2
<div class="z-container"> <div class="z-box box1">Box 1</div> <div class="z-box box2">Box 2</div> </div> .box1 { z-index: 1; } .box2 { z-index: 2; }

3. CSS Transform

The transform property applies rotation, scaling, skewing, or translation to elements.

<div class="transform-box"></div> .transform-box:hover { transform: rotate(20deg) scale(1.2) translateX(20px); }

4. CSS Filter

The filter property applies visual effects such as blur, grayscale, and brightness.

<img src="image.jpg"> img:hover { filter: grayscale(80%) blur(2px) brightness(1.2); }

5. CSS Perspective

The perspective property defines the 3D depth view for transformed elements.

<div class="perspective-container"> <div class="perspective-box"></div> </div> .perspective-container { perspective: 600px; } .perspective-box { transform: rotateY(25deg); }
