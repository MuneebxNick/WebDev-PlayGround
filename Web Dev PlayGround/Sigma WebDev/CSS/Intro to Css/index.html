Welcome to CSS

What is CSS?

CSS is a stylesheet language that describes the presentation of a document written in HTML or XML.

It allows you to apply styles to web pages, making them visually appealing.

Why Use CSS?

CSS helps separate content from design, making it easier to maintain and update web pages.

It also allows for consistent styling across multiple pages.

Basic CSS Syntax

CSS rules consist of a selector and a declaration block.

A selector targets an HTML element, and the declaration block contains one or more declarations.

Each declaration includes a property and a value, separated by a colon.

For example:

selector {
        property: value;
    }

Common CSS Properties

  • color: Changes the text color.
  • background-color: Changes the background color.
  • font-size: Changes the size of the text.
  • border: Adds a border around an element.
  • margin: Adds space outside an element.
  • padding: Adds space inside an element.
