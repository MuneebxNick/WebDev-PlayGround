More ON CSS Selectors

CSS selectors are powerful tools for selecting HTML elements to apply styles. Here are some advanced selectors:

  • Universal Selector (*): Selects all elements in the document.
  • Type Selector (element): Selects all elements of a specific type, e.g., p for paragraphs.
  • Class Selector (.classname): Selects all elements with a specific class.
  • ID Selector (#idname): Selects a single element with a specific ID.
  • Attribute Selector ([attribute]): Selects elements with a specific attribute, e.g., [type="text"].
  • Pseudo-classes (:pseudo-class): Selects elements based on their state, e.g., :hover, :focus.
  • Pseudo-elements (::pseudo-element): Selects parts of an element, e.g., ::before, ::after.
  • Descendant Selector (ancestor descendant): Selects elements that are descendants of a specified ancestor.
  • Child Selector (parent > child): Selects direct children of a specified parent.
  • Sibling Selector (previous + next): Selects the next sibling of a specified element.
  • General Sibling Selector (previous ~ siblings): Selects all siblings that follow a specified element.
  • Group Selector (selector1, selector2): Applies the same styles to multiple selectors.
  • Attribute Selector with Value ([attribute="value"]): Selects elements with a specific attribute value.
  • Negation Pseudo-class (:not(selector)): Selects elements that do not match a specified selector.
  • First Child Selector (:first-child): Selects the first child of a parent element.
  • Last Child Selector (:last-child): Selects the last child of a parent element.
  • Nth Child Selector (:nth-child(n)): Selects the nth child of a parent element, where n can be a number, keyword, or formula.
  • Attribute Selector with Multiple Values ([attribute~="value"]): Selects elements with an attribute that contains a specific word.
  • Attribute Selector with Starts With ([attribute^="value"]): Selects elements with an attribute that starts with a specific value.
  • Attribute Selector with Ends With ([attribute$="value"]): Selects elements with an attribute that ends with a specific value.
  • Lang Selector (:lang(language)): Selects elements based on the language specified in the lang attribute.
  • Only Child Selector (:only-child): Selects an element that is the only child of its parent.
  • Only-of-type Selector (:only-of-type): Selects an element that is the only one of its type among its siblings.
  • Empty Selector (:empty): Selects elements that have no children (including text nodes).
  • Checked Selector (:checked): Selects input elements that are checked, such as checkboxes or radio buttons.

These selectors can be combined and used in various ways to create complex styles for your web pages. Understanding how to use them effectively can greatly enhance your CSS skills.
