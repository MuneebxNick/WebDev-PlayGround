CSS Selectors

CSS selectors are patterns used to select the elements you want to style. They can be simple or complex, allowing you to target specific elements based on their type, class, ID, attributes, and more. Here are some common CSS selectors:

with defination

Common CSS Selectors

  • Universal Selector: *
  • Type Selector: div
  • Class Selector: .className
  • ID Selector: #idName
  • Attribute Selector: [attribute]
  • Pseudo-class Selector: :hover
  • Pseudo-element Selector: ::before
  • Descendant Selector: div p
  • Child Selector: div > p
  • Adjacent Sibling Selector: h1 + p
  • General Sibling Selector: h1 ~ p
  • Group Selector: h1, h2, h3
  • Attribute Selector with Value: [type="text"]
  • Attribute Selector with Partial Match: [class^="btn-"]
  • Attribute Selector with Substring Match: [class*="btn"]
  • Attribute Selector with End Match: [class$="-active"]
  • Negation Pseudo-class: :not(.className)
  • First Child Selector: :first-child
  • Last Child Selector: :last-child
  • Nth Child Selector: :nth-child(2n)
  • Nth Last Child Selector: :nth-last-child(2n)
  • First of Type Selector: :first-of-type
  • Last of Type Selector: :last-of-type
  • Nth of Type Selector: :nth-of-type(2n)
  • Only Child Selector: :only-child
  • Only of Type Selector: :only-of-type
  • Empty Selector: :empty
  • Enabled Selector: :enabled
  • Disabled Selector: :disabled
  • Checked Selector: :checked
  • Required Selector: :required
  • Optional Selector: :optional
  • Valid Selector: :valid
  • Invalid Selector: :invalid
  • Focus Selector: :focus

These selectors can be combined and used in various ways to create powerful and specific styles for your web pages.

Example Usage


/* Universal Selector */
* {
    margin: 0;
    padding: 0;
}
/* Type Selector */
div {
    background-color: lightblue;
}
/* Class Selector */
.className {
    color: red;
}
/* ID Selector */
#idName {
    font-size: 20px;
}
/* Attribute Selector */
[attribute] {
    border: 1px solid black;
}
/* Pseudo-class Selector */
a:hover {
    text-decoration: underline;
}
/* Pseudo-element Selector */
a::before {
    content: "Link: ";
}
/* Descendant Selector */
div p {
    color: green;
}
/* Child Selector */
div > p {
    font-weight: bold;
}
/* Adjacent Sibling Selector */
h1 + p {
    margin-top: 10px;
}
/* General Sibling Selector */
h1 ~ p {
    color: gray;
}
/* Group Selector */
h1, h2, h3 {
    font-family: Arial, sans-serif;
}
/* Attribute Selector with Value */
[type="text"] {
    background-color: yellow;
}
/* Attribute Selector with Partial Match */
[class^="btn-"] {
    padding: 10px;
    background-color: blue;
    color: white;
}
/* Attribute Selector with Substring Match */
[class*="btn"] {
    border-radius: 5px;
}
/* Attribute Selector with End Match */
[class$="-active"] {
    background-color: green;
}
/* Negation Pseudo-class */
:not(.className) {
    opacity: 0.5;
}
/* First Child Selector */
div:first-child {
    border: 2px solid red;
}
/* Last Child Selector */
div:last-child {
    border: 2px solid blue;
}
/* Nth Child Selector */

div:nth-child(2n) {
    background-color: lightgray;
}
/* Nth Last Child Selector */
div:nth-last-child(2n) {
    background-color: lightcoral;
}
/* First of Type Selector */
div:first-of-type {
    font-style: italic;
}
/* Last of Type Selector */
div:last-of-type {
    font-weight: bold;
}
/* Nth of Type Selector */
div:nth-of-type(2n) {
    background-color: lightgreen;
}
/* Only Child Selector */
div:only-child {
    padding: 20px;
    background-color: lightyellow;
}
/* Only of Type Selector */
div:only-of-type {
    border: 1px solid black;
}
/* Empty Selector */
div:empty {
    display: none;
}
/* Enabled Selector */
input:enabled {
    background-color: white;
}
/* Disabled Selector */
input:disabled {
    background-color: lightgray;
}
/* Checked Selector */
input:checked {
    background-color: lightblue;
}
/* Required Selector */
input:required {
    border: 2px solid red;
}
/* Optional Selector */
input:optional {
    border: 2px solid green;
}
/* Valid Selector */
input:valid {
    border: 2px solid green;
}
/* Invalid Selector */
input:invalid {
    border: 2px solid red;
}
/* Focus Selector */
input:focus {
    outline: 2px solid blue;
}

These examples demonstrate how to use various CSS selectors to style HTML elements effectively.

Conclusion

CSS selectors are powerful tools that allow you to target and style specific elements on your web pages. By understanding and using these selectors, you can create visually appealing and well-structured designs.
