Overflow Property - CSS

The overflow property in CSS controls how content that overflows an element's box is handled. It can take the following values:

  • visible: Content is not clipped and may overflow the element's box.
  • hidden: Content is clipped and not visible outside the element's box.
  • scroll: Content is clipped, but a scrollbar is added to view the overflowing content.
  • auto: If content overflows, a scrollbar is added; otherwise, no scrollbar is shown.
  • clip: Similar to hidden, but does not allow scrolling.

Example Usage

div {
    width: 200px;
    height: 100px;
    overflow: auto; /* Adds a scrollbar if content overflows */
    
}

Related Properties

  • overflow-x: Controls horizontal overflow.
  • overflow-y: Controls vertical overflow.
  • overflow-wrap: Controls how text wraps within an element.
  • text-overflow: Controls how overflowed content is displayed in a single line.

Browser Compatibility

The overflow property is widely supported across all modern browsers. However, specific behaviors may vary slightly, especially with the auto value.
