Transform Property - CSS

The transform property in CSS allows you to apply various transformations to an element, such as rotation, scaling, translation, and skewing.

Syntax

transform: none | transform-function | transform-function-list;

Values

  • none: No transformation is applied.
  • translate(x, y): Moves the element from its current position.
  • scale(x, y): Scales the element by the specified factors.
  • rotate(angle): Rotates the element by the specified angle.
  • skew(x-angle, y-angle): Skews the element along the X and Y axes.
  • matrix(a, b, c, d, e, f): Applies a 2D transformation using a matrix.
  • perspective(value): Applies a perspective transformation to the element.
  • transform-function-list: A comma-separated list of transform functions.

Examples

Translate Example

div {
    transform: translate(50px, 100px);
}

Scale Example

div {
    transform: scale(1.5, 2);
}

Rotate Example

div {
    transform: rotate(45deg);
}

Skew Example

div {
    transform: skew(20deg, 10deg);
}

Matrix Example

div {
    transform: matrix(1, 0, 0, 1, 50, 100);
}

Perspective Example

div {
    transform: perspective(500px) rotateY(45deg);
}

Browser Compatibility

The transform property is widely supported in modern browsers. However, for older browsers, you may need to use vendor prefixes like -webkit-, -moz-, and -ms-.

Conclusion

The transform property is a powerful tool in CSS that allows you to manipulate the appearance of elements on the page. By combining different transform functions, you can create complex visual effects and animations.
