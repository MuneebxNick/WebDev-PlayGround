Filters in CSS

CSS filters allow you to apply graphical effects like blurring or color shifting to an element.

CSS Filter Properties

  • blur(px): Applies a Gaussian blur to the element.
  • brightness(%): Adjusts the brightness of the element.
  • contrast(%): Adjusts the contrast of the element.
  • drop-shadow(offsetX offsetY blurRadius color): Applies a shadow effect to the element.
  • grayscale(%): Converts the element to grayscale.
  • hue-rotate(deg): Applies a hue rotation to the element.
  • invert(%): Inverts the colors of the element.
  • opacity(%): Adjusts the opacity of the element.
  • saturate(%): Adjusts the saturation of the element.
  • sepia(%): Applies a sepia filter to the element.

Usage

To apply a filter, use the filter property in your CSS. For example:

        .example {
            filter: blur(5px);
        }

Multiple filters can be combined by separating them with spaces:

        .example {
            filter: blur(5px) brightness(150%);
        }

Note that not all filters are supported in all browsers, so check compatibility before using them.
