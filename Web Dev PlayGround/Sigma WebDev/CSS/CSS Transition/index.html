Transition in CSS

CSS transitions allow you to change property values smoothly (over a given duration).

Example

Hover over me!

Properties

  • transition-property: Specifies the name of the CSS property to which the transition is applied.
  • transition-duration: Defines how long the transition should take.
  • transition-timing-function: Describes how the intermediate values of the CSS properties being affected by a transition effect are calculated.
  • transition-delay: Specifies when the transition effect will start.

Browser Support

CSS transitions are supported in all modern browsers. However, for older browsers, you may need to use vendor prefixes like -webkit-, -moz-, etc.

Conclusion

CSS transitions provide a simple way to create smooth animations and effects on web pages, enhancing user experience.
