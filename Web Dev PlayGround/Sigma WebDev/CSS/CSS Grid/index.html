Grid - CSS

CSS Grid Layout is a two-dimensional layout system for the web. It allows you to create complex layouts with rows and columns, making it easier to design responsive web pages.

Grid Properties

  • grid-template-columns: Defines the number and size of columns in the grid.
  • grid-template-rows: Defines the number and size of rows in the grid.
  • gap: Sets the space between rows and columns.
  • grid-area: Allows you to assign a specific area for an item within the grid.
  • justify-items: Aligns items horizontally within their grid area.
  • align-items: Aligns items vertically within their grid area.
  • grid-auto-flow: Controls how items are placed in the grid when there is extra space.
  • grid-template-areas: Allows you to define named grid areas for easier layout management.
