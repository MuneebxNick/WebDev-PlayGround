Outlines & Shadows

Outlines and shadows are used to enhance the appearance of elements on a webpage.

Outlines

Outlines are lines that are drawn around elements, similar to borders but do not take up space.

Shadows

Shadows can be applied to elements to create a sense of depth.

CSS Properties

Here are the CSS properties used for outlines and shadows:

  • outline: Defines the outline of an element.
  • box-shadow: Applies shadow effects around an element.
  • outline-color: Sets the color of the outline.
  • outline-style: Specifies the style of the outline (e.g., solid, dashed).
  • outline-width: Sets the width of the outline.
  • box-shadow-offset-x, box-shadow-offset-y: Define the horizontal and vertical offsets of the shadow.
  • box-shadow-blur-radius: Sets the blur radius of the shadow.
  • box-shadow-spread-radius: Defines the spread radius of the shadow.
  • box-shadow-color: Sets the color of the shadow.

HTML Code Example

        
            <div style="outline: 2px solid blue; padding: 10px; margin: 10px;">
                This is an element with an outline.
            </div>
            <div style="box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5); padding: 20px; margin: 10px;">
                This is an element with a shadow.
            </div>
            <div style="outline: 2px solid red; box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5); padding: 20px; margin: 10px;">
                This element has both an outline and a shadow.
            </div>

CSS Code Example

        
            .outline-example {
                outline: 2px solid blue;
                padding: 10px;
                margin: 10px;
            }
            .shadow-example {
                box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);
                padding: 20px;
                margin: 10px;
            }
            .combined-example {
                outline: 2px solid red;
                box-shadow: 5px 5px 10px rgba(0, 0, 0, 0.5);
                padding: 20px;
                margin: 10px;
            }

Conclusion

Using outlines and shadows effectively can greatly enhance the visual appeal of your web elements.
