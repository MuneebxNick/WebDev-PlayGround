List Style Types

CSS provides various list style types to customize the appearance of lists. Here are some common types:

  • disc - Default bullet style.
  • circle - Hollow circle bullet.
  • square - Square bullet.
  • decimal - Numbered list (1, 2, 3...).
  • lower-alpha - Lowercase letters (a, b, c...).
  • upper-alpha - Uppercase letters (A, B, C...).
  • lower-roman - Lowercase Roman numerals (i, ii, iii...).
  • upper-roman - Uppercase Roman numerals (I, II, III...).

Example of List Styling

ul {
    list-style-type: square;
    padding-left: 20px;
}
li {
    margin-bottom: 10px;
}

Customizing List Styles

You can also use custom images for list items:

ul.custom-list {
    list-style-image: url('path/to/image.png');
    padding-left: 20px;
}
li.custom-list-item {
    margin-bottom: 10px;
}

Nested Lists

Lists can be nested to create sub-lists:

  • Main Item 1
    • Sub Item 1.1
    • Sub Item 1.2
  • Main Item 2
    • Sub Item 2.1
    • Sub Item 2.2

Conclusion

Styling lists in CSS allows you to create visually appealing and organized content. By using different list style types and customizing them, you can enhance the user experience on your web pages.
