CSS Display Property

The display property in CSS is used to control the layout of elements on a web page. It determines how an element is displayed in the document flow.

Common Display Values

  • block: The element is displayed as a block-level element, taking up the full width available.
  • inline: The element is displayed as an inline element, only taking up as much width as necessary.
  • inline-block: The element is displayed as an inline-level block container, allowing it to have width and height.
  • none: The element is not displayed at all (it has no effect on layout).
  • flex: The element becomes a flex container, enabling flexible layouts.
  • grid: The element becomes a grid container, allowing for grid-based layouts.
  • table: The element is displayed as a table.
  • table-row: The element is displayed as a table row.
  • table-cell: The element is displayed as a table cell.
  • list-item: The element is displayed as a list item.
  • run-in: The element behaves like a block-level element, but can also be treated as an inline element in certain contexts.

Example Usage

/* Block-level element */
div {
    display: block;
}
/* Inline element */
span {
    display: inline;
}
/* Inline-block element */
span.inline-block {
    display: inline-block;
}
/* Flex container */
.container {
    display: flex;
}
/* Grid container */
.grid-container {
    display: grid;
}
/* Hidden element */
.hidden {
    display: none;
}

Notes

Using the display property effectively can greatly enhance the layout and responsiveness of your web pages. It is essential to understand how different display values interact with each other, especially when using flexbox or grid layouts.
