Operators in JavaScript

JavaScript provides various operators to perform operations on variables and values. The most common types of operators are:

  • Arithmetic Operators: Used to perform mathematical operations.
  • Assignment Operators: Used to assign values to variables.
  • Comparison Operators: Used to compare two values.
  • Logical Operators: Used to combine multiple conditions.

1. Arithmetic Operators

    let a = 10;
    let b = 5;

    console.log(a + b); // Addition
    console.log(a - b); // Subtraction
    console.log(a * b); // Multiplication
    console.log(a / b); // Division
    console.log(a % b); // Modulus

2. Assignment Operators

    let x = 10; // Assign
    x += 5;     // Add and assign
    console.log(x); // 15
    x *= 2;     // Multiply and assign
    console.log(x); // 30

3. Comparison Operators

    let num1 = 10;
    let num2 = '10';

    console.log(num1 == num2); // true (loose equality)
    console.log(num1 === num2); // false (value and type check ----- strict equality)
    console.log(num1 != num2); // false (loose inequality)
    console.log(num1 !== num2); // true (strict inequality)

4. Logical Operators

    let condition1 = true;
    let condition2 = false;

    console.log(condition1 && condition2); // false (AND)
    console.log(condition1 || condition2); // true (OR)
    console.log(!condition1); // false (NOT)

If, Else If, Else Ladder

The if, else if, else ladder is used to execute different blocks of code based on multiple conditions. It allows you to check for various conditions sequentially and execute the corresponding block of code for the first true condition.

Syntax

    if (condition1) {
    // block of code to be executed if condition1 is true
    } else if (condition2) {
    // block of code to be executed if condition1 is false and condition2 is true
    } else {
    // block of code to be executed if both condition1 and condition2 are false
    }

Example

let score = 85;
if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else if (score >= 70) {
    console.log("Grade: C");
} else if (score >= 60) {
    console.log("Grade: D");
} else {
    console.log("Grade: F");
}

In this example, the variable score is checked against multiple conditions using an if-else if-else ladder. Depending on the value of score, the appropriate grade is printed to the console.

Notes

  • You can have multiple else if statements to check for different conditions.
  • The else statement is optional and can be omitted if not needed.
  • Conditions are evaluated in order, and the first true condition will execute its corresponding block of code.
  • If none of the conditions are true, the code inside the else block will be executed (if it exists).

Ternary Operator

The ternary operator is a shorthand way of writing an if-else statement. It takes three operands: a condition, a value to return if the condition is true, and a value to return if the condition is false.

    condition ? value_if_true : value_if_false;

Example

let age = 18;
let canVote = (age >= 18) ? "Yes, you can vote." : "No, you cannot vote.";
console.log(canVote);

In this example, the ternary operator checks if the variable age is greater than or equal to 18. If true, it assigns "Yes, you can vote." to the variable canVote; otherwise, it assigns "No, you cannot vote."

Notes

  • The ternary operator is a concise way to write simple conditional statements.
  • It is often used for assigning values based on a condition.
  • For complex conditions or multiple branches, it is better to use if-else statements for readability.
