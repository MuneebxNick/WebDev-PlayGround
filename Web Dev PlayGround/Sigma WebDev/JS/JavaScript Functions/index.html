JavaScript Functions

Functions are blocks of code designed to perform a particular task. They are executed when "called" or "invoked".

Function Declaration

function greet(name) {
    return "Hello, " + name + "!";
}

Function Invocation

console.log(greet("Alice")); // Outputs: Hello, Alice!

Function Expression

const add = function(a, b) {
    return a + b;
};

Arrow Function

const multiply = (a, b) => a * b;

Example Usage

console.log(add(5, 3)); // Outputs: 8
console.log(multiply(4, 2)); // Outputs: 8

Function with Default Parameters

function greet(name = "Guest") {
    return "Hello, " + name + "!";
}

Example Usage with Default Parameter

console.log(greet()); // Outputs: Hello, Guest!

Function with Rest Parameters

function sum(...numbers) {
    return numbers.reduce((total, num) => total + num, 0);
}

Example Usage with Rest Parameters

console.log(sum(1, 2, 3, 4)); // Outputs: 10
