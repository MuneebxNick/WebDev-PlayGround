JavaScript Variables

Variables are containers for storing data values. In JavaScript, you can declare a variable using var, let, or const.

Types of Variables

  • var: Declares a variable that can be re-assigned.
  • let: Declares a block-scoped variable that can be re-assigned.
  • const: Declares a block-scoped variable that cannot be re-assigned.

            var name = "John"; // Using var
            let age = 30;     // Using let
            const pi = 3.14;  // Using const

Note: It's a good practice to use let and const over var to avoid scope-related issues.

Rules For Naming A Variables

  • Variable names must begin with a letter, underscore (_), or dollar sign ($).
  • Subsequent characters can also include digits (0-9).
  • Variable names are case-sensitive (e.g., myVar and myvar are different).
  • Reserved words (like let, const, if, etc.) cannot be used as variable names.
  • Use meaningful names that describe the purpose of the variable.

Example of valid variable names:


            let firstName;
            const _age;
            var $salary;
            let user1;

Example of invalid variable names:


            let 1stName; // Invalid: starts with a digit
            const my-age; // Invalid: contains a hyphen
            var let; // Invalid: reserved word

Global Vs Scoped Variables

Global Variables: Declared outside any function or block, accessible from anywhere in the code.


            var globalVar = "I am global";

            function showGlobal() {
                console.log(globalVar); // Accessible here
            }

            showGlobal();
            console.log(globalVar); // Accessible here too

Scoped Variables: Declared within a function or block, accessible only within that function or block.


            function myFunction() {
                let scopedVar = "I am scoped";
                console.log(scopedVar); // Accessible here
            }

            myFunction();
            console.log(scopedVar); // Error: scopedVar is not defined

Using let and const creates block-scoped variables, meaning they are only accessible within the block they are defined in (e.g., within an if statement or a loop).


            if (true) {
                let blockVar = "I am block scoped";
                console.log(blockVar); // Accessible here
            }

            console.log(blockVar); // Error: blockVar is not defined

Data Types

JavaScript has several data types, including:

  • String: Represents text, e.g., "Hello, World!"
  • Number: Represents numeric values, e.g., 42
  • Boolean: Represents a logical entity and can have two values: true or false.
  • Object: A collection of properties, e.g., { name: "John", age: 30 }
  • Array: A special type of object for storing ordered collections, e.g., [1, 2, 3]
  • Null: Represents the intentional absence of any object value.
  • Undefined: A variable that has been declared but has not yet been assigned a value.

Objects

Objects are a key part of JavaScript and allow you to store collections of data and more complex entities.


            const person = {
                name: "Alice",
                age: 25,
                greet: function() {
                    console.log("Hello, " + this.name);
                }
            };
            person.greet(); // Outputs: Hello, Alice

In this example, person is an object with properties name and age, and a method greet.
