Loops - JavaScript

Loops can execute a block of code as long as a specified condition is true.

Types of Loops

  • for - loops through a block of code a number of times
  • for/in - loops through the properties of an object
  • for/of - loops through the values of an iterable object
  • while - loops through a block of code while a specified condition is true
  • do/while - also loops through a block of code while a specified condition is true

Examples

for loop

for (let i = 0; i < 5; i++) {
    console.log(i);
}

for/in loop

const person = {fname: "John", lname: "Doe", age: 25};
for (let key in person) {
    console.log(key + ": " + person[key]);
}

for/of loop

const arr = [10, 20, 30, 40, 50];
for (let value of arr) {
    console.log(value);
}

while loop

let i = 0;
while (i < 5) {

    console.log(i);
    i++;
}

do/while loop

let i = 0;
do {
    console.log(i);
    i++;
} while (i < 5);

Nested Loops

Nested loops are loops inside another loop. They are useful for working with multi-dimensional data structures. 

for (let i = 0; i < 3; i++) {
for (let j = 0; j < 2; j++) {
    console.log(`i: ${i}, j: ${j}`);
}
}

Loop Control Statements

Control statements can alter the flow of loops:

  • break - exits the loop immediately.
  • continue - skips the current iteration and continues with the next one.

Using break

for (let i = 0; i < 5; i++) {
if (i === 3) {
    break; // exits the loop when i is 3
}
console.log(i);
}

Using continue

for (let i = 0; i < 5; i++) {
if (i === 2) {
    continue; // skips the iteration when i is 2
}
console.log(i);
}

Looping Through Arrays

Loops are commonly used to iterate over arrays. Here are some examples:

Using for loop

const fruits = ["Apple", "Banana", "Cherry"];
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

Using for/in loop

const fruits = ["Apple", "Banana", "Cherry"];
for (let index in fruits) {
    console.log(index + ": " + fruits[index]);
}

Using for/of loop

const fruits = ["Apple", "Banana", "Cherry"];
for (let fruit of fruits) {
    console.log(fruit);
}

Using while loop

const fruits = ["Apple", "Banana", "Cherry"];
let i = 0;
while (i < fruits.length) {
    console.log(fruits[i]);
    i++;
}

Using do/while loop

const fruits = ["Apple", "Banana", "Cherry"];
let i = 0;
do {
    console.log(fruits[i]);
    i++;
} while (i < fruits.length);
